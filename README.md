# 🐹 がんばれハム子 DX

猫の手から逃げながら、制限時間内にアイテムを集めるブラウザアクションゲーム。
Power Apps Code App 版（`FightHamuko`）をベースに、純粋なブラウザゲーム（Vite + React、Power Apps 依存なし）として作り直した DX 版です。

**🎮 [ゲームをプレイする](https://minoru365.github.io/FightHamukoDX/)**

## 遊び方

- **移動**: 矢印キー / WASD キー、または画面をドラッグして仮想ジョイスティック操作
- **目的**: 制限時間内にすべてのアイテム（🥕など）を集めるとステージクリア
- **敵**: 猫の手に触れるとゲームオーバー（🛡️シールド中は弾く）
- 全 **5 ステージ**。進むほど敵が増え・速くなる

## DX 版の強化ポイント

- **ステージ進行**: 全5ステージ。クリアごとに残り時間ボーナス＋難易度上昇。最終ステージ突破で ALL CLEAR
- **パワーアップ**（フィールドに出現、取得で発動）
  - 🛡️ シールド（無敵で敵を弾く） / ⚡ スピードアップ / 🧲 マグネット（アイテム吸引） / 🕸️ スロー（敵減速） / ⏰ タイム +6秒
- **敵パターン**: 直線追跡 (chaser) / 先読み (ambush) / 巡回 (patrol)
- **障害物**: 🪨🌳🌵🍄 が移動を妨げる（押し出し判定）
- **演出**: パーティクル、フローター（獲得点表示）、画面シェイク、オーラ、カウントダウン、コンボ表示
- **コンボ・スコア**: 連続取得でボーナス（+50 × コンボ）
- **サウンド**: Web Audio API による効果音＋手続き的 BGM（🔊 でミュート切替）
- **ハイスコア保存**: 難易度別に localStorage へ保存

## 難易度

| 難易度 | アイテム | 制限時間 | 敵速度 | 敵数 |
|--------|---------|---------|--------|------|
| 😊 Easy   | 8〜  | 90秒 | 2.0 | 1 |
| 🙂 Normal | 12〜 | 70秒 | 2.8 | 1 |
| 😰 Hard   | 16〜 | 55秒 | 3.6 | 2 |
| 👹 鬼     | 20〜 | 40秒 | 4.4 | 2 |

※ ステージが進むとアイテム数・敵数・敵速度が増加します。

## 開発

```powershell
npm install      # 依存インストール
npm run dev      # 開発サーバー (http://localhost:3000/)
npm run build    # 本番ビルド (dist/)
npm run preview  # ビルド結果のプレビュー
npm run lint     # ESLint
```

## 技術スタック

- React 19 + TypeScript + Vite 7
- 固定タイムステップ (1/60秒) の requestAnimationFrame ゲームループ
- 描画は DOM（`<img>` / `<div>`）ベース、`World` を ref で保持して毎フレーム再描画

## ディレクトリ構成

```
src/
  App.tsx, main.tsx, index.css
  assets/images/   キャラ・敵・アイテム・背景
  game/
    Game.tsx       メインコンポーネント＆ゲームループ
    Game.css       ゲーム画面のスタイル・アニメーション
    types.ts       型定義（World / Player / Enemy など）
    audio.ts       サウンドエンジン（SFX + BGM）
    storage.ts     ハイスコア保存（localStorage）
```
